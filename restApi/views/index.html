Landsat on AWS

Landsat on AWS makes Landsat data available for anyone to use via Amazon S3. All Landsat 8 scenes from 2015 are available along with a selection of cloud-free scenes from 2013 and 2014. All new Landsat 8 scenes are made available each day, often within hours of production.

This site allows you to browse all Landsat imagery available on AWS.

About Landsat

The Landsat program is a joint effort of the U.S. Geological Survey and NASA and is the longest running program to gather Earth imagery from space.

Learn more at NASA’s Landsat Science site.

Accessing Landsat on AWS

All data is available in the landsat-pds Amazon S3 bucket in the in US West (Oregon) AWS Region. Data is optimized for programattic access from Amazon EC2.

More information on accessing Landsat on AWS.

Earth on AWS

A variety of Earth observation and environmental datasets are available at no cost on AWS. Analyze data without needing to download or store your own copy.

Learn more about Earth on AWS.

This is a Serverless Site

This site uses AWS Lambda and Amazon API Gateway to create web pages on demand without maintaining any servers.

Learn more on the AWS Compute blog.
